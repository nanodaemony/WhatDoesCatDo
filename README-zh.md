
### 主要特性
- 集成spring boot 常用开发组件集、公共配置、AOP日志等
- 单体架构，更快更简单
- 提供PC管理端与APP端代码接口
- 集成mybatis plus快速dao操作
- 快速生成后台代码: entity/dto/query/vo/controller/service/mapper/xml
- 集成Swagger/Knife4j，可自动生成api文档
- 集成Redis缓存
- 集成HikariCP连接池，JDBC性能和慢查询检测
